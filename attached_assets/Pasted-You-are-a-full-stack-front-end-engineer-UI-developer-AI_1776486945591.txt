Storage helpers (exact file: src/lib/storage.ts)

- Exported helpers:
  - getItem(key: string): T | null
  - setItem(key: string, value: T): void
  - removeItem(key: string): void
  - hydrateFromSeed(seedPath: string, storageKey: string): Promise
- Behavior:
  - hydrateFromSeed reads the corresponding seed JSON in /src/data/ and writes it to localStorage only if the storageKey is empty.
